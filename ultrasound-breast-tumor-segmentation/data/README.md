# Data Directory

This directory is a placeholder. The datasets used in this project are **not included** in the repository due to their size and licensing terms.

## Download Instructions

### BUSI Dataset (~150 MB)

1. Go to [Kaggle — Breast Ultrasound Images Dataset](https://www.kaggle.com/datasets/aryashah2k/breast-ultrasound-images-dataset)
2. Download `archive.zip`
3. Rename it to `archive_segmentation.zip`
4. Place it in `data/raw/`

### BUS-BRA Dataset (~2 GB)

1. Go to [Zenodo — BUS-BRA](https://doi.org/10.5281/zenodo.8231412) and request access
2. Download `BUSBRA.zip`
3. Place it in `data/raw/`

The pipeline will auto-detect and extract both ZIPs on first run.

## Expected Layout After Download

```
data/
├── README.md           ← this file
├── raw/
│   ├── archive_segmentation.zip   ← BUSI
│   └── BUSBRA.zip                 ← BUS-BRA (optional)
├── processed/          ← auto-generated by the pipeline
└── samples/            ← small sample images for testing (included)
```

## Sample Images

A small set of anonymized sample images is provided in `data/samples/` for testing the inference pipeline without downloading the full datasets.
